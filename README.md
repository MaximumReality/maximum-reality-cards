# Maximum Reality Playing Cards

A custom, original playing card deck created for a Maximum Reality–themed solitaire game.

## Deck Structure
- Standard 52-card deck
- Custom card back
- Aces feature Lori as the Ace face
- Face cards (J, Q, K) are custom designs
- Number cards (2–10) use original artwork
- One Joker featuring Azul as the wild card

## File Naming Convention
All files use lowercase names with underscores:

- a_hearts.png
- 2_hearts.png
- k_spades.png
- joker.png
- card-back.png
